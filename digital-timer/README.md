# Digital Timer

## 📌 Description
A real-time digital timer application that dynamically displays the current time using JavaScript. This project demonstrates working with time functions and DOM updates.

🔗 **Live Demo:**  
https://Naveen-Kumar45.github.io/top-JS-projects/digital-timer/

## 🚀 Features
- Displays real-time clock (hours, minutes, seconds)
- Updates every second dynamically
- Clean and minimal UI
- Lightweight and fast

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 💡 What I Learned
- JavaScript Date object
- setInterval() for real-time updates
- DOM manipulation

## ▶️ How to Run
Open index.html in your browser
